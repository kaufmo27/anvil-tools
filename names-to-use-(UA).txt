------------------------------------------------
Anvil Tools: Працюючі Назви
------------------------------------------------
-Тут ви знайдете всі назви предметів для зміни зовнішнього вигляду.
-Коли перейменовуєте в грі, ігноруйте знак "+" та лапки, аби все працювало як слід. Наприклад: Аметистове Кайло, Меч Енду, Мотика з Черепом
------------------------------------------------
Інструменти:
Аметистовий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
Кістяний(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
Медовий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий/Залізний Меч, Кайло, Сокира, Лопата, Мотика)
Льодовий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
Мятний(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
Зарослий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика + Лук і Арбалет).
Руіновий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика + Лук і Арбалет).
Скалковий(-а, -е) + "Назва Інструмента" (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
"Назва Інструмента" + Пломеня (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
"Назва Інструмента" + Енду (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика + Лук і Арбалет)
"Назва Інструмента" + Гідності (Працює для Незеритовий/Діамантовий/Залізний Меч, Кайло, Сокира, Лопата, Мотика)
"Назва Інструмента" + з Черепом (Працює для Незеритовий/Діамантовий Меч, Кайло, Сокира, Лопата, Мотика).
Магічний Лук (Працює лише з луком)
Заморожений Меч (Працює лише для Незеритового/Діамантового Меча)
